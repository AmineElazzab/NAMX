# NamX API

## This API has the following ressources :

- SIGN IN
- SIGN UP
- TYPE VEHICULE
- OPTION DU TYPE VEHICULE
- PRE-ORDER
- ADMIN (BACK-OFFICE)
- CONFIRMATION BY EMAIL

## Available Scripts

### `cd namx`
### `npm install`
### `npm start`

### `cd api`
### `yarn`
### `yarn dev`
